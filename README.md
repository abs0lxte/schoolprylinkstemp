# External HTML v.0.98
=====================
A repository for pulling external links/text from GitHub directly to static HTML.
This project remains until a suitable alternative is found or static HTML usage is deprecated.

## Info 
External HTML Text/Link Hub -----------------------------
Description
This repository serves as a hub for collecting external links and text from GitHub, converting them to static HTML.

## Usage
You can use it as needed, anythiing important can be pulled via raw github user,
